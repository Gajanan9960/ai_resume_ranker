About AI Resume Ranker

AI Resume Ranker is a powerful resume screening platform designed for recruiters and hiring teams. It uses AI and natural language processing to automatically parse, analyze, and rank candidate resumes based on job-specific skills and relevance.

✨ Features
  • Automatic parsing and skill extraction from PDF/image resumes.
  • Semantic matching using Sentence-BERT and curated skill synonyms.
  • Tag-based dashboard filtering, smart scoring, and top candidate badges.
  • Support for CSV/Excel export of filtered data.
  • Admin login, job title filters, and real-time candidate insights.
🧠 How It Works

Uploaded resumes are processed using OCR (for images) and NLP pipelines to extract skills and experiences. These are semantically compared to job requirements using transformer-based models. The result is a ranked list of candidates optimized for relevance and technical match.
