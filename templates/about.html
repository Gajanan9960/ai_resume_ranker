About Resume Ranker AI

Resume Ranker AI is an intelligent recruitment tool designed to streamline the hiring process. By leveraging advanced Natural Language Processing (NLP) and Machine Learning algorithms, it automatically analyzes resumes against job descriptions to identify the most suitable candidates.

Key Features

  • Automated Scoring: Instantly scores resumes based on relevance to the job description.
  • Skill Matching: Extracts and compares technical skills to ensure candidate qualifications match requirements.
  • Semantic Analysis: Uses sentence transformers to understand the context and meaning behind the text, not just keyword matching.
  • Efficient Filtering: Sort and filter candidates by score, status, or job role to quickly find top talent.
  • OCR Support: Upload scanned resume images — text is extracted automatically using Tesseract OCR.
  • Email Integration: Send tailored emails (acceptance, interview invite, rejection) directly from the dashboard.

How It Works

  1. Upload: Recruiters upload candidate resumes (PDF/Image) and provide a job description.
  2. Analyze: The system extracts text, identifies skills, and computes a semantic similarity score.
  3. Rank: Candidates are ranked and categorized (Top Candidate, Shortlisted, etc.) based on their scores.
  4. Review: Recruiters can view detailed results, filter candidates, and export data for further processing.

Scoring Methodology

Each resume receives a weighted composite score (0–100%) calculated from four components:

Component Weight Description
Keyword Match 40% Ratio of matched job keywords found in the resume text
Semantic Similarity 40% Cosine similarity between resume and job description embeddings (Sentence-BERT)
Keyword Density 10% Density of relevant keywords throughout the resume
Resume Quality 10% Based on word count — optimal range is 300–2000 words

Bonuses & penalties: +10% bonus if keyword match ≥ 70%; −15% penalty if < 20%.

Tag Thresholds

Tag Score Range
Top Candidate ≥ 80%
Shortlisted 65% – 79%
Review 45% – 64%
Rejected < 45%

Tech Stack

  • Backend: Python, Flask
  • NLP: spaCy (en_core_web_sm), Sentence-BERT (all-MiniLM-L6-v2)
  • OCR: pytesseract + Pillow
  • Database: MongoDB (pymongo)
  • Export: pandas + XlsxWriter (Excel), CSV
  • Email: Flask-Mail (SMTP)
  • Deployment: Gunicorn, Render
